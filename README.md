# AIEarth-Internship

## 基于AIEarth平台的多源遥感影像地物分类与植被指数时序分析

技术栈：AIEarth平台、WorldView-3多光谱影像（1.24m）、哨兵2号影像（10m）、AIEC中国10米地物分类数据集、NDVI指数计算、真彩色合成（RGB-B4B3B2）、渲染与卷帘对比分析

具体工作：导入2018年WorldView-3多光谱影像，在AIEarth平台执行自动地物分类；导入2020年AIEC天紫湖地区10米分类结果，并与WorldView-3结果进行对比；导入天子湖地区2018年12月及2022年四季（3月、7月、10月、12月）哨兵2号影像，设置真彩色波段并调整拉伸范围；对四季哨兵影像批量计算NDVI，并按统一阈值进行渲染及分段标签设置；利用卷帘工具对比四季NDVI变化，并分析不同分辨率影像的分类差异原因。

量化成果：生成 2幅 WorldView-3精细分类图，识别出 4类 地物（草地、林地、建筑、耕地），建筑等细小地物可被有效检出；对比显示：WorldView-3（1.24m）较AIEC（10m）分辨率提升约 8倍，后者无法识别建筑类目标；完成 4个季度 的NDVI时序制图，定量揭示春夏NDVI显著高于秋冬的季节性规律（渲染色阶由红转绿）；卷帘对比实现 4期影像 同步可视化，辅助快速识别植被覆盖度变化。
